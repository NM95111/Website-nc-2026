# Nicolas & Cecilia — Wedding Website

A static wedding website for Nicolas & Cecilia, hosted on GitHub Pages.

## Pages

- **Welcome** (`index.html`) — Hero, invitation, itinerary, venue info
- **Our Story** (`our-story.html`) — Photo timeline of the couple's journey
- **Travel & Accommodation** (`travel.html`) — Lebanon travel guide, restaurants & tips
- **Our Registry** (`registry.html`) — Bank details for honeymoon fund contributions
- **RSVP** (`rsvp.html`) — Guest RSVP form (powered by Formspree)

## Tech Stack

- Pure HTML5 / CSS3 / Vanilla JavaScript — no build step required
- Google Fonts: Great Vibes, Cormorant Garamond, Montserrat
- Hosted via GitHub Pages (static, served from repo root)

## Local Preview

Open `index.html` directly in a browser — no server needed.
